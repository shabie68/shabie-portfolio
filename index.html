My problem solving approach

Problem solving is one of my core characteristics. I often inspect problems from differnet angles or try doing the same thing in different ways to sharpen my problem solving skills. I am continuously working on it as I do the following to further sharpen my problem solving skills. When solving any problem I use the following method.

  • Reread the problem multiple times
  • Write the problem to articulate my thoughts
  • I just try to solve the problem
  • When a problem is solved I then brainstorm for different solutions and then refactor accordingly
